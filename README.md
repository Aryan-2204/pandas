# 🐼 Python Pandas Learning Journey

Welcome to my **Pandas learning repository**!
This repo documents my journey of learning and practicing **data analysis using Python Pandas** — from basics to advanced concepts.

##  About Pandas

**Pandas** is a powerful Python library used for:

* Data manipulation
* Data analysis
* Data cleaning
* Working with structured datasets (like CSV, Excel, SQL)
